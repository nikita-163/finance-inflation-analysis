# Finance Inflation Analysis

## О проекте

В рамках этого проекта анализируются личные расходы, их структура, динамика и возможная связь с инфляцией и ключевой ставкой Центрального Банка Российской Федерации. 

## Бизнес-задачи

В рамках проекта необходимо ответить на следующие вопросы:

- как менялись личные расходы в течение года?
- какие категории расходов занимают наибольшую долю бюджета?
- изменялся ли средний чек со временем?
- наблюдается ли сезонность в расходах?
- насколько рост расходов соответствует уровню инфляции?
- какой вклад в снижение фактических расходов вносит кэшбэк?

## Цели проекта

- Создать базу данных для хранения банковских операций.
- Загрузить исходные данные в PostgreSQL.
- Очистить и подготовить данные с помощью SQL.
- Провести исследовательский анализ данных (EDA) с помощью Python + pandas.
- Проанализировать структуру и динамику расходов.
- Сопоставить изменение расходов с показателями инфляции.
- Построить наглядные визуализации.
- Подготовить аналитический отчет с основными выводами.

## Источники данных

В проекте используются два набора данных:

- личная банковская выписка (около 560 операций);
- ежемесячные данные Банка России по инфляции и ключевой ставке.

Примечание: исходная банковская выписка не размещается в репозитории, поскольку содержит персональные финансовые данные.

## Стек технологий

- PostgreSQL — хранение данных.
- DBeaver — работа с базой данных.
- SQL — загрузка, очистка и анализ данных.
- Python + pandas — обработка и анализ данных.
- Matplotlib — визуализация результатов.
- Git — контроль версий.
